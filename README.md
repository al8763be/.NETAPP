Script kitty guide to how to push software like a sane person 

1. Hitta detta repo (yay!)
2. kör: git clone https://github.com/al8763be/.NETAPP.git
3. Profit? 

När du vill pusha till main 
git add . (För att lägga till dina ändringar till stage)
git commit -m "Beskriving av vad du gjort så jag fattar vad det är"
git remote add origin https://github.com/al8763be/.NETAPP.git 
git push origin 

får du error så är det antagligen för att du inte har du inte en github ssh key 
tar två sekunder att fixa
